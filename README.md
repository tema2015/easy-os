# Easy OS

Easy OS — простой Linux для новичков.
Основан на Debian + XFCE.

## Как получить ISO
1. Fork репозиторий
2. Открой Actions
3. Запусти "Build Easy OS ISO"
4. Скачай ISO

## Возможности
- Работает в VirtualBox
- Live режим
- Установка на ПК
- Мышь и клавиатура из коробки
- Понятный интерфейс
